	Working with strings 💭️💭️💭️💭️💭️💭️

	phrase = "My name is Kaung Khant Zaw")
	puts ("My name is \"Kaung Khant Zaw\" ")					> backslech(\) is used to omit the ", newline in the strings.
	puts phrase.upcase()										> makes uppercase letter in the phrase
	puts phrase.downcase()										> makes lowercase letter in the phrase
	puts phrase.strip()											> remove unnecessary spaces 
	puts phrase.length()										> the number of characters in the phrase
	puts phrase.include? "Zaw"									> is Zaw in the phrase?
	puts phrase[11]												> print out 'K' from the phrase
	puts phrase[11,15]											> print out 'Kaung Khant Zaw' because [11] is starting point and [15] is the number of index amount.
	puts phrase.index("K")										> print the index of K
	puts "kaung khant zaw".upcase()								> KAUNG KHANT ZAW
	puts "KAUNG KHANT ZAW".downcase()							> kaung khant zaw

	Getting Input from the user 💭️💭️💭️💭️💭️💭️💭️

	puts "Enter Your Name : "
	name = gets.chomp()											> gets is getting input from user

	age = gets.chomp()											> and chomp() is used to rid off the new line when we clicked enter
	puts ("Your name is "+ name + " and your age is "+ age)

	puts "Enter a number "	
	num1 = gets.chomp().to_i									> get input ,remove new line and convert into interger	
	puts "Enter another number "								> but '.to_i' can't calculate floating numbers
	num2 = gets.chomp().to_i	
	result = num1 + num2 
	puts result

	puts "Enter a number "	
	num1 = gets.chomp().to_f									> get input, remove new line and convert into floating number  
	puts "Enter another number "								> .to_f can calculate all types of numbers 
	num2 = gets.chomp().to_f	
	result = num1 + num2 
	puts result
